# D1

> Source: https://sst.dev/docs/component/cloudflare/d1/

## Overview

The `D1` component integrates a Cloudflare D1 database into your SST application, enabling serverless SQL database functionality within the Cloudflare ecosystem. D1 is Cloudflare's native serverless SQL database built on SQLite.

## Constructor

```typescript
new D1(name, args?, opts?)
```

**Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | `string` | Yes | Component identifier |
| `args` | `D1Args` | No | Configuration options |
| `opts` | `ComponentResourceOptions` | No | Pulumi resource options |

## D1Args (Constructor Props)

### transform

- **Type:** `Object`
- **Required:** No
- **Description:** Customizes underlying resource creation.

| Sub-property | Type | Description |
|--------------|------|-------------|
| `database` | `D1DatabaseArgs \| (args, opts, name) => void` | Customize the D1 database resource creation |

```typescript
const db = new sst.cloudflare.D1("MyDatabase", {
  transform: {
    database: {
      // Custom D1 database args
    }
  }
});
```

## Properties

### databaseId

- **Type:** `Output<string>`
- **Description:** The generated ID of the D1 database.

### nodes

- **Type:** `Object`
- **Description:** The underlying Pulumi resources.

| Sub-property | Type | Description |
|--------------|------|-------------|
| `database` | `D1Database` | The Cloudflare D1 database resource |

## SDK Integration

### Links

Accessible through the `Resource` object from the `sst` package:

- `databaseId` (`string`) - The D1 database identifier

### Bindings

Once linked to a worker, the database becomes queryable using Cloudflare's D1 client API. You can use the `prepare` method to create prepared statements and execute SQL queries.

```typescript
import { Resource } from "sst";

// Execute a query
const result = await Resource.MyDatabase.prepare(
  "SELECT id FROM todo ORDER BY id DESC LIMIT 1"
).first();

// Insert data
await Resource.MyDatabase.prepare(
  "INSERT INTO todo (title, completed) VALUES (?, ?)"
).bind("My task", false).run();

// Query with parameters
const rows = await Resource.MyDatabase.prepare(
  "SELECT * FROM todo WHERE completed = ?"
).bind(true).all();
```

## Examples

### Basic Setup

```typescript
const db = new sst.cloudflare.D1("MyDatabase");
```

### Link to a Worker

```typescript
const db = new sst.cloudflare.D1("MyDatabase");

new sst.cloudflare.Worker("MyWorker", {
  handler: "./index.ts",
  link: [db],
  url: true
});
```

### Accessing D1 in Worker Code

```typescript
import { Resource } from "sst";

export default {
  async fetch(request) {
    // Query the database
    const result = await Resource.MyDatabase.prepare(
      "SELECT id FROM todo ORDER BY id DESC LIMIT 1"
    ).first();

    return new Response(JSON.stringify(result));
  }
};
```

### Using Prepared Statements

```typescript
import { Resource } from "sst";

export default {
  async fetch(request) {
    // Insert with bound parameters
    await Resource.MyDatabase.prepare(
      "INSERT INTO users (name, email) VALUES (?, ?)"
    ).bind("John", "john@example.com").run();

    // Query all rows
    const { results } = await Resource.MyDatabase.prepare(
      "SELECT * FROM users"
    ).all();

    return new Response(JSON.stringify(results));
  }
};
```
