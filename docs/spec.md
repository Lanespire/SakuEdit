# SakuEdit 仕様書

## 概要

SakuEditは「急いで動画編集したい人・編集分からない人でもサクッと編集できる」AI動画編集ツールです。参考動画のスタイルをAIが学習し、字幕・カット・テンポを自動で再現します。

### コンセプト
- **ターゲットユーザー**: 動画編集に不慣れなユーザー、素早く編集を完了したいユーザー
- **主要機能**: 無音トリミング、字幕・文字起こし、テンポ再現、YouTuberスタイル分析・再現
- **技術スタック**: Next.js 16.1.5、React 19.2.3、Tailwind CSS 4、Remotion

### カラーテーマ
| 色 | 値 | 用途 |
|-----|------|------|
| Primary | `#f97415` | メインアクションボタン、強調表示 |
| Background Light | `#fcfaf8` | ライトモード背景 |
| Background Dark | `#23170f` | ダークモード背景 |
| Text Main | `#1c130d` | メインテキスト |
| Text Sub | `#9e6b47` | サブテキスト |

### 1. ランディング/アップロード画面

#### UI要素
- **ヘッダー**: ロゴ（🎬 アイコン + 「SakuEdit」）、ナビゲーション（編集/スタイル学習/履歴）、ログインリンク、PROボタン
- **メインコピー**: 「この人みたいな動画、作れます」「参考動画のスタイルをAIが学習。字幕・カット・テンポを自動で再現します。」
- **編集する動画セクション**:
  - ドロップ/クリックアップロードエリア（青枠線、ドロップハイライト）
  - YouTube/TikTok URL入力フィールド
- **参考動画セクション**（PRO機能）:
  - 「PRO」バッジ
  - ドロップ/クリックアップロードエリア（オレンジ枠線）
  - YouTube URL入力フィールド
  - 例ボタン（HIKAKIN、東海オンエア、QuizKnock）
- **プリセット選択**: 横スクロールプリセットカード（YouTuber定番、Shorts/リール、ビジネス、Vlog、ゲーム実況）
- **オプションチェックボックス**: 字幕生成、無音カット、テンポ最適化、BGM提案（PRO）
- **「AIで編集をはじめる」ボタン**: 大型グラデーションボタン
- **フッター情報**: 平均処理時間、動画保存保証、月3本まで無料

#### 機能
- 動画ファイルのドラッグ＆ドロップまたはクリックでのアップロード
- YouTube/TikTok URLからの動画取得
- 参考動画のアップロードまたはURL指定（PRO）
- プリセットスタイルの選択
- 編集オプションのON/OFF切り替え
- 編集処理の開始

#### 状態
- **初期状態**: 入力フィールドが空、全オプションON
- **アップロード中**: アップロードエリアにプログレス表示
- **バリデーションエラー**: ファイルサイズ超過、対応外フォーマット

- → スタイル選択画面（プリセット選択時）
- → スタイル分析中画面（参考動画指定時）

### 5. 編集画面内の処理中ステータス（動画処理）

#### UI要素
- **ヘッダー**: プロジェクト名、「キャンセル」ボタン

  - 時間表示
  - 顔検出ボックス（シミュレーション）

- **右側: ステータス＆ステップ**（45%）:
  - **進捗セクション**:
    - パーセンテージ表示（78%）
    - 「Processing」バッジ
    - メッセージ（「あなたの動画にスタイルを適用中...」）
    - 残り時間表示
    - 円形プログレスインジケーター（SVG）

  - **処理ステップ一覧**:
    - 動画を読み込み（完了）
    - 音声から字幕を生成 42/45（完了）
    - 参考スタイルを適用中...（処理中）:
      - 字幕デザイン適用（完了）
      - カットポイント検出（処理中）
      - テンポ調整（待機）
    - 無音区間をカット（待機）
    - 最終レンダリング（待機）
  - **Live Log**:
    - タイムスタンプ
    - アニメーションタイプ（FadeIn/Bounce/Slide/Plain/Pop）
    - 字幕テキスト

#### 機能
- 編集画面に入る前段として、同一URL上で処理進捗を表示
- 元動画の実プレビュー表示
- DBの `progress` / `progressMessage` に基づく進捗確認
- 処理ステップとライブログ確認
- 完了後にそのまま編集UIへ切り替え

#### 状態

## ユーザーフロー

### メインフロー
```
1. ランディング画面
   ├── 動画をアップロード or URL入力
   ├── 参考動画を指定（オプション・PRO）
   ├── プリセットを選択 or スタイルを分析
   └──「AIで編集をはじめる」

2. スタイル選択画面（プリセット/クリエイター/URL/カスタム）
   ├── プリセットを検索・フィルタリング
   ├── クリエイタースタイルを選択
   ├── YouTube URLからスタイルを分析
   └──「適用する」

3. スタイル分析中画面（URLから学習時のみ）
   ├── 参考動画のプレビュー
   ├── 分析進捗を確認
   └── 分析完了

4. 編集画面内の処理中ステータス
   ├── 実動画プレビュー
   ├── 処理ステップを確認
   ├── Live Logで進捗を確認
   └── 処理完了後そのまま編集UIへ切替

5. 編集画面（メイン画面）
   ├── ビデオプレビュー
   ├── タイムラインで編集
   ├── AI提案を適用
   ├── 字幕を編集（モーダル）
   └──「書き出し」

6. 書き出し設定モーダル
   ├── 解像度を選択
   ├── フォーマットを選択
   ├── 字幕出力方法を選択
   └──「書き出しを開始」

7. マイプロジェクト画面（履歴）
   ├── プロジェクト一覧を確認
   ├── フィルタリング
   └── 既存プロジェクトの編集・再開
```

### 補足フロー
- **料金プラン画面**: → PRO購入 → 機能解放
- **ヘッダーナビゲーション**: 編集/スタイル学習/履歴/料金ページ間遷移

## 共通コンポーネント

### ナビゲーションバー
- **要素**: ロゴ、タブナビゲーション、PROボタン、ユーザーアバター
- **スタイル**: グラスモーフィズム、アクティブタブは背景色
- **機能**: ページ遷移、ログイン状態表示

### モーダル
- **スタイル**: ドロップダウンまたはボトムシート（モバイル）
- **構成**: ヘッダー（タイトル + 閉じるボタン）、コンテンツ（スクロール可能）、フッター（キャンセル/保存ボタン）
- **アニメーション**: フェードイン + スライドアップ

### プログレスバー
- **種類**:
  - リニアプログレス（処理画面）
  - 円形プログレス（スタイル分析中）
  - ステッププログレス（処理画面のステップリスト）

### カード
- **スタイル**: 角丸、ボーダー、シャドウ、ホバー時のエフェクト
- **種類**: プロジェクトカード、スタイルカード、プランカード、AI提案カード

### トグル/スイッチ
- **スタイル**: iOS風スイッチ（オフ/オン）
- **用途**: オプションのON/OFF切り替え

### ドロップダウン
- **スタイル**: Tailwind Forms使用、カスタム矢印アイコン
- **用途**: フォーマット選択、スタイルプリセット選択

### 通知/トースト
- **位置**: 画面下部中央または右上
- **種類**: 成功（緑）、エラー（赤）、情報（青）、警告（オレンジ）

### ローディングスピナー
- **種類**:
  - デフォルトスピナー（`animate-spin`）
  - スロースピナー（`animate-spin-slow`）
  - パルス（`animate-pulse`）

### バッジ
- **種類**: PRO、推奨、おすすめ、人気急上昇
- **スタイル**: プライマリーカラー、小サイズ、大文字

## アニメーション

### 定義済みアニメーション
```css
@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
.animate-spin {
  animation: spin 1s linear infinite;
}
.animate-spin-slow {
  animation: spin 3s linear infinite;
}
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
.animate-pulse {
  animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;
}
```

### トランジション
- **標準**: `transition-all duration-200`
- **ホバー**: `transition-colors duration-300`
- **スケール**: `transform hover:scale-110`
