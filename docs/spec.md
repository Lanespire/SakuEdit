# SakuEdit 仕様書

## 概要

SakuEditは「急いで動画編集したい人・編集分からない人でもサクッと編集できる」AI動画編集ツールです。参考動画のスタイルをAIが学習し、字幕・カット・テンポを自動で再現します。

### コンセプト
- **ターゲットユーザー**: 動画編集に不慣れなユーザー、素早く編集を完了したいユーザー
- **主要機能**: 無音トリミング、字幕・文字起こし、テンポ再現、YouTuberスタイル分析・再現
- **技術スタック**: Next.js 16.1.5、React 19.2.3、Tailwind CSS 4、Remotion

### カラーテーマ
| 色 | 値 | 用途 |
|-----|------|------|
| Primary | `#f97415` | メインアクションボタン、強調表示 |
| Background Light | `#fcfaf8` | ライトモード背景 |
| Background Dark | `#23170f` | ダークモード背景 |
| Text Main | `#1c130d` | メインテキスト |
| Text Sub | `#9e6b47` | サブテキスト |

### 6. 結果 + React Video Editor Pro 編集画面（メイン画面）

#### UI要素
- **トップバー**: プロジェクト名、プロジェクト一覧への導線、RVE の保存状態/操作群
- **左サイドバー**: RVE のデフォルトサイドバー。メディア、テキスト、音声、図形、トランジションなどの追加導線
- **中央プレビューエリア**:
  - RVE canvas プレビュー
  - 再生/停止、シーク、ズーム、フルスクリーンなどの再生コントロール
  - 現在フレーム/時間表示
- **右側詳細パネル**: 選択中オーバーレイのプロパティ編集
- **下部タイムライン**:
  - RVE Pro の advanced timeline
  - 動画/音声/テキスト/図形/エフェクト等の複数トラック
  - ドラッグ移動、リサイズ、ズーム、再生ヘッド同期
- **書き出し導線**:
  - RVE から `/api/latest/ssr` を使った Remotion レンダリング
  - 書き出し後のダウンロード導線

#### 機能
- `Project.compositionData` に `kind: rve-pro` のドキュメントを保存し、編集内容を自動保存する
- `compositionData` が空の既存 project は source video から RVE 初期 state を生成して開く
- タイムライン操作（スクロール、ズーム、クリップ移動・リサイズ、選択同期）
- プレビュー再生とタイムライン/キャンバスの同期
- Remotion SSR 経由の書き出し
- 処理中は同一 `/edit/[id]` で進捗 UI を表示し、完了後にそのまま RVE UI へ切り替える
- 当面は既存字幕編集モーダルや caption パネルとの後方互換は持たず、RVE 置き換えを優先する

#### 状態
- **再生中**: プレビュー再生、再生ヘッド移動
- **編集中**: RVE 上でオーバーレイ・タイムラインを編集
- **自動保存中**: RVE state を IndexedDB と `Project.compositionData` に同期
- **書き出し中**: SSR render の進捗を確認

#### 遷移
- → RVE 内の各編集パネル（要素選択時）
- → 書き出し処理/ダウンロード導線（書き出し時）

## ユーザーフロー

### メインフロー
```
1. ランディング画面
   ├── 動画をアップロード or URL入力
   ├── 参考動画を指定（オプション・PRO）
   ├── プリセットを選択 or スタイルを分析
   └──「AIで編集をはじめる」

2. スタイル選択画面（プリセット/クリエイター/URL/カスタム）
   ├── プリセットを検索・フィルタリング
   ├── クリエイタースタイルを選択
   ├── YouTube URLからスタイルを分析
   └──「適用する」

3. スタイル分析中画面（URLから学習時のみ）
   ├── 参考動画のプレビュー
   ├── 分析進捗を確認
   └── 分析完了

4. 編集画面内の処理中ステータス
   ├── 実動画プレビュー
   ├── 処理ステップを確認
   ├── Live Logで進捗を確認
   └── 処理完了後そのまま編集UIへ切替

5. 編集画面（メイン画面）
   ├── ビデオプレビュー
   ├── タイムラインで編集
   ├── AI提案を適用
   ├── 字幕を編集（モーダル）
   └──「書き出し」

6. 書き出し設定モーダル
   ├── 解像度を選択
   ├── フォーマットを選択
   ├── 字幕出力方法を選択
   └──「書き出しを開始」

7. マイプロジェクト画面（履歴）
   ├── プロジェクト一覧を確認
   ├── フィルタリング
   └── 既存プロジェクトの編集・再開
```

### 補足フロー
- **料金プラン画面**: → PRO購入 → 機能解放
- **ヘッダーナビゲーション**: 編集/スタイル学習/履歴/料金ページ間遷移

## 共通コンポーネント

### ナビゲーションバー
- **要素**: ロゴ、タブナビゲーション、PROボタン、ユーザーアバター
- **スタイル**: グラスモーフィズム、アクティブタブは背景色
- **機能**: ページ遷移、ログイン状態表示

### モーダル
- **スタイル**: ドロップダウンまたはボトムシート（モバイル）
- **構成**: ヘッダー（タイトル + 閉じるボタン）、コンテンツ（スクロール可能）、フッター（キャンセル/保存ボタン）
- **アニメーション**: フェードイン + スライドアップ

### プログレスバー
- **種類**:
  - リニアプログレス（処理画面）
  - 円形プログレス（スタイル分析中）
  - ステッププログレス（処理画面のステップリスト）

### カード
- **スタイル**: 角丸、ボーダー、シャドウ、ホバー時のエフェクト
- **種類**: プロジェクトカード、スタイルカード、プランカード、AI提案カード

### トグル/スイッチ
- **スタイル**: iOS風スイッチ（オフ/オン）
- **用途**: オプションのON/OFF切り替え

### ドロップダウン
- **スタイル**: Tailwind Forms使用、カスタム矢印アイコン
- **用途**: フォーマット選択、スタイルプリセット選択

### 通知/トースト
- **位置**: 画面下部中央または右上
- **種類**: 成功（緑）、エラー（赤）、情報（青）、警告（オレンジ）

### ローディングスピナー
- **種類**:
  - デフォルトスピナー（`animate-spin`）
  - スロースピナー（`animate-spin-slow`）
  - パルス（`animate-pulse`）

### バッジ
- **種類**: PRO、推奨、おすすめ、人気急上昇
- **スタイル**: プライマリーカラー、小サイズ、大文字

## アニメーション

### 定義済みアニメーション
```css
@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
.animate-spin {
  animation: spin 1s linear infinite;
}
.animate-spin-slow {
  animation: spin 3s linear infinite;
}
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
.animate-pulse {
  animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;
}
```

### トランジション
- **標準**: `transition-all duration-200`
- **ホバー**: `transition-colors duration-300`
- **スケール**: `transform hover:scale-110`
