# SakuEdit LP 素材発注メモ

最終更新: 2026-03-14

## 目的

LP の「何もはまっていない」箇所を、生成アセットで順次置き換えるための発注メモ。
現状の [app/page.tsx](/Users/takahashimotoki/product/sakuedit/app/page.tsx) はコード描画のモックが多いため、以下の優先順で素材を差し込む。

## 方針

- 動きで理解させたい箇所は `Sora 2` の短尺動画
- 説明補助や比較表示は画像生成
- UI 上に重ねる見出し・CTA は後から実装側で載せるので、アセット内に文字は焼き込まない
- 人物やPC画面は日本語圏のクリエイター文脈に寄せる
- 縦横両対応を考え、被写体は中央寄せ、上下左右 10% は安全余白にする

## 優先度A

### 1. Hero キービジュアル

- 配置先: [app/page.tsx](/Users/takahashimotoki/product/sakuedit/app/page.tsx) の `EditorMockup`
- 用途: 「アップロードしたら編集がほぼ終わる」を一目で伝える主ビジュアル
- 種別: 画像生成
- サイズ: 1600x1000 以上
- 比率: 16:10
- 依頼プロンプト:

```text
日本語の動画編集SaaSのヒーロー用キービジュアル。ノートPCに動画編集UIが表示されており、字幕トラック、無音カットのマーカー、書き出し完了に近い状態が一目でわかる。雰囲気はクリーン、信頼感、少し温かみ。白背景ベースのLPに合う。実写寄りではなくプロダクトビジュアル寄り。文字は入れない。被写体は中央。UIは日本語前提。
```

- 受け入れ条件:
  - 字幕とカット処理が同時に視認できる
  - 暗すぎず、LP の白背景で埋もれない
  - 画面内の文字が判読不能なノイズになっていない

### 2. 処理フローデモ

- 配置先: [app/page.tsx](/Users/takahashimotoki/product/sakuedit/app/page.tsx) の `LandingProcessingDemo`
- 用途: アップロード後に自動処理が進む体験の可視化
- 種別: `Sora 2` 動画
- 長さ: 6-8 秒
- 比率: 16:9
- 依頼プロンプト:

```text
SaaS LP 用の短いUIデモ動画。日本語の動画編集サービスで、動画アップロード後に「音声解析」「字幕生成」「無音カット」「サムネイル生成」が順番に進み、最後に編集画面が開く。カメラ移動は最小限。クリーンで高級感のあるプロダクトデモ。文字は最小限、ローディングと進捗が視覚的に伝わる。ループしやすい自然な終わり方。
```

- 受け入れ条件:
  - ループ再生で違和感がない
  - 進行中から完了までが 1 本で理解できる
  - 手元や顔の演技ではなく、プロダクト体験が主役

- 種別: 画像生成
- サイズ: 1600x1000 以上
- 比率: 16:10
- 依頼プロンプト:

```text
日本語の話し手動画フレームに、視認性の高い字幕が載っているプロダクト用ビジュアル。字幕は大きく、タイムコード付き編集を連想させる。字幕スタイル違いを比較できるUI要素も少し見える。教育系またはビジネス系の動画トーン。文字の焼き込みは最小限。白背景のLPに合うよう、全体は暗すぎない。
```

- 受け入れ条件:

  - 日本語動画の文脈に見える

## 優先度B

### 4. サムネイル候補セクション

- 配置先: [app/page.tsx](/Users/takahashimotoki/product/sakuedit/app/page.tsx) の `feature-thumbnail`
- 用途: 1 本の動画から複数サムネ候補が出ることを見せる
- 種別: 画像生成
- サイズ: 1600x1000 以上
- 比率: 16:10
- 依頼プロンプト:

```text
動画編集SaaS内でサムネイル候補が4案表示されているUIビジュアル。1案が選択状態。YouTube向けの日本語動画サムネイルらしい強い視認性。人物の表情、余白、コントラストがあり、クリックしたくなる。アプリUIの一部として自然に見える。文字は読めなくてよいので焼き込みは控えめ。
```

  - 1案が選択済みとわかる

- 比率: 16:10

## 運用メモ

- 先に作る順番は `1 -> 2 -> 3 -> 4 -> 5 -> 6`
- Hero と処理フローだけ先に差し替えても LP の印象は大きく改善する
- 実プロダクトのスクリーンショットが揃ったら、生成素材は順次差し替える
- 生成素材に UI テキストを焼き込むと差し替えコストが高いので避ける
